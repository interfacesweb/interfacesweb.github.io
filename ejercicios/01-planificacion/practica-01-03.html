1 Introducción

En esta práctica realizaremos un prototipo de interfaz web utilizando el programa Pencil, disponible en este enlace para su descarga. Se intentarán seguir las recomendaciones dadas en la teoría para garantizar un buen nivel de usabilidad.

Si no se desea utilizar Pencil puede utilizarse cualquier otro programa similar (balsamiq, wireframe.cc,…).

2 Tiempo asignado

3 horas de clase.

3 Requisitos del prototipo

Simularemos que estamos realizando un prototipo para un sitio web que tenga 3 niveles de navegación. Se deberá realizar el prototipo de cada uno de los tipos de página enunciados a continuación (un diseño para cada tipo). Cada uno de los tipos de página deberá contener una serie de elementos, que se muestran también a continuación.

3.1 Tipos de página

  • Página de inicio (home page).
  • Página de primer nivel.
  • Página de segundo nivel.
  • Página de tercer nivel.
  • Página con formulario de alta / registro.
  • Página con información de contacto / localización.
  • Página con formulario de contacto.

3.2 Elementos que deben mostrarse

En cada una de las páginas deberán mostrarse los elementos necesarios para cumplir con la función que tengan asignada así como los elementos de diseño apropiados para organizar visualmente el contenido: capas, cabecera, menús, pies de página, elementos de formulario, encabezados, enlaces,…

El diseño visual y la disposición de las capas, así como el tipo de menús (desplegables, en línea, etc.) queda a criterio del alumno.

A continuación se listan algunos elementos adicionales que deben aparecer en el sitio web. Se debe decidir en qué página aparecerá cada uno de ellos (es posible que varios de ellos aparezcan en varias páginas). Se deben seguir las recomendaciones estudiadas en la teoría.

  • Logo de la empresa.
  • Nombre de la empresa (el nombre de la empresa contendrá las iniciales del alumno).
  • Título de la página.
  • Cuadro de búsqueda.
  • Menú de navegación principal.
  • Menú de navegación secundario.
  • Menú de navegación terciario.
  • Mapa de localización.
  • Dirección y teléfono de la empresa.
  • Botón de Inicio (Home).
  • Ofertas o mensajes destacados.
  • Enlace para el registro / inicio de sesión.
  • Banners de anuncios.

4 Resultado a entregar

  • Se deberá exportar cada uno de los diseños en formato PNG.
  • Se creará un archivo ZIP con todos los archivos PNG.
  • El nombre del archivo a entregar será de la forma apellido1-apellido2-nombre-practica3.zip (por ejemplo, prieto-alarcon-pedro-practica3.zip).
