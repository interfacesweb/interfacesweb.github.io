1 Introducción

Se pretende realizar la maquetación de un sitio web formado por 4 páginas. Este sitio trata sobre la historia de la informática. Cada una de dichas páginas trata sobre los acontecimientos más importantes que se produjeron en el campo de la informática en un siglo concreto.

2 Tiempo asignado

2 horas de clase.

3 Desarrollo de la práctica

Se debe modificar el código de cada una de las páginas para transformarlas en páginas HTML con las siguientes características:

  • Deben tener todas las etiquetas propias del formato HTML 5 (html, head, body, title,…). En este enlace puedes encontrar un esquema de documento HTML 5 básico.
  • La página tendrá una sección denominada outer que englobará todo el contenido de la página.
  • La sección outer debe dividirse a su vez en cuatro secciones: cabecera, menú, cuerpo y pie de página. Cada una de ellas estará identificada con un identificador distinto (que será el mismo para todas las páginas) y contendrá la información correspondiente. Las secciones se realizarán utilizando etiquetas DIV.
  • El contenido debe organizarse en una estructura acorde con el texto. Para ello se utilizarán encabezados (pueden ser de distintos niveles) y párrafos.
  • Cada página debe disponer de un menú que permita navegar entre cada una de las 4 páginas que componen el sitio. El menú se diseñará como una lista desordenada.
  • Cada página debe disponer de una cabecera donde se muestre el título de la página.
  • Cada página debe disponer de un pie de página donde se muestre vuestro nombre y apellidos y el año actual.
  • Cada página debe tener remarcardos tres trozos de texto no consecutivos mediante una etiqueta span. Dichos trozos de texto deben estar contenidos en un párrafo (es decir, no se puede seleccionar texto de dos o más párrafos distintos). Los textos remarcados deben estar identificados por un nombre de clase llamado "destacado".

5 Resultado a entregar

  • Se deben entregar los cuatro ficheros HTML comprimidos en un sólo archivo.
  • El archivo estará en formato ZIP.
  • El nombre del archivo será de la forma apellido1-apellido2-nombre-practica4.zip (por ejemplo, prieto-alarcon-pedro-practica4.zip).
