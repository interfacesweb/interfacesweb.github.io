1 Introducción

En ocasiones puede ser interesante no reinventar la rueda a la hora de diseñar un sitio web. Existen multitud de sitios que ofrecen plantillas hechas con HTML y CSS que pueden ser de utilidad para nuestros diseños. A continuación veremos las características de dichas plantillas y algunos sitios web donde descargarlas.

2 Plantillas

2.1 Características

Las plantillas pueden estar compuestas por archivos HTML, CSS, imágenes y scripts (fundamentalmente JavaScript). Suelen incluir textos predefinidos para mostrar su aspecto visual. Lo más habitual es que se distribuyan en archivos comprimidos.

El documento HTML suele estar estructurado en capas y utiliza nombres de clase e identificadores para modificar el aspecto visual con CSS.
