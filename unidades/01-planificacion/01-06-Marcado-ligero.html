1 Introducción

Editar código HTML puede resultar tedioso por la cantidad de etiquetas y símbolos que conlleva. Por ello se han desarrollado una serie de lenguajes para crear documentos

2 Markdown

Markdown es uno de los lenguajes de marcado ligero más populares. Se utiliza en numerosos sitios web por su facilidad y poco peso.

2.1 Sintaxis

La sintaxis de Markdown puede consultarse en este enlace: http://daringfireball.net/projects/markdown/syntax

2.2 Editor online

Dillinger (http://dillinger.io/) es un editor online para Markdown. Permite editar texto en este formato y exportar el resultado a HTML. De esta manera se puede realizar un prototipo de página de manera rápida y sencilla.
